Natural Language Processing COVID Attitudes Tracker


THE POINT


what it does

It crawls Twitter for current tweets, uses deep learning NLP to classify them, and builds a cute little map to visualize our data

how it does it

This bad boy uses Tweepy to acquire tweets, AllenNLP + Pytorch for model training and deployment, and Flask for backend, and CSS/HTML/Bootstrap/JS therein to deploy this website you're reading.
